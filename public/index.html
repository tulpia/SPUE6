Webpack du groupe d'auto-eval

Instructions

  • npm run watch permet de compiler le code en sauvegardant grâce au Hot Module Replacement

  • npm run dev compile le code sans minifier

  • npm run build compile le code en le minifiant, pour une prochaine mise en ligne

  • npm install nomdevotrepackage --save-dev permet d'installer un module et de le sauvegarder en tant dépendance de développeur
    Le module ne sera donc pas inclus dans le code final (quand vous allez faire npm run build)

  • npm install nomdevotrepackage --save permet d'inclure un module dans le code final, c'est très utile pour des modules comme jquery, axios, gsap, etc...

  • npm run lint permet de "linter" le code, c'est à dire de le laver, en quelques sortes.
    Le linter inclu suit les styles guides de Airbnb
